#### 注册   / register 

|     字段     | 是否必填 |   用途   |         码值         |   备注   |
| :----------: | :------: | :------: | :------------------: | :------: |
|   userName   |    是    |  用户名  |                      |          |
| userPassword |    是    |   密码   |                      |          |
|  userMobile  |    是    |  手机号  |                      |          |
|  userEmail   |    是    |   邮箱   |                      |          |
|  createTime  |    否    | 注册时间 |                      |          |
|   userSex    |    是    |   性别   | 1:男，2：女，3：未知 |          |
|  userStatus  |    否    | 用户状态 |   0：失效，1：生效   |          |
|    userId    |    否    |  用户id  |                      | 随机生成 |
